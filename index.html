The Cart API~

Coupon System with a few rules and discount types, Where new rules and discount types added dynamically.

Cart API

  • GET All Items From Cart: http://localhost:8080/api/carts

  • ADD New Item To Cart: http://localhost:8080/api/carts/save 

                {
                "name": "mirror",
                "price": 7000
            }

  • DELETE Item From Cart: http://localhost:8080/api/carts/{id}

  • CheckOut Coupon On Total Cart Item: http://localhost:8080/api/carts/coupon 

                {
                "coupon":"MIXED10"
            }
Coupon Rules API

  • ADD New Coupon Rule: http://localhost:8080/api/carts/save 

                {
                "name": "MIXED10",
                "priceRange": 200,
                "percentOff": 10,
                "amountOff": 10,
                "limit": 3
           }

  • DELETE Coupon Rule: http://localhost:8080/api/carts/{id}

  • GET Coupon Rules: http://localhost:8080/api/carts/{id}
