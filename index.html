The Cart API~

Coupon System with a few rules and discount types, Where new rules and discount types added dynamically.

Cart API

  • GET All Items From Cart: https://dynamic-cart-api.herokuapp.com/api/carts

  • ADD New Item To Cart: https://dynamic-cart-api.herokuapp.com/api/carts/save 

                {
                "name": "mirror",
                "price": 7000
            }

  • DELETE Item From Cart: https://dynamic-cart-api.herokuapp.com/api/carts/{id}

  • CheckOut Coupon On Total Cart Item: https://dynamic-cart-api.herokuapp.com/api/carts/coupon 

                {
                "coupon":"MIXED10"
            }
Coupon Rules API

  • GET Coupon Rules: https://dynamic-cart-api.herokuapp.com/api/coupons

  • ADD New Coupon Rule: https://dynamic-cart-api.herokuapp.com/api/coupons/save 

                {
                "name": "MIXED10",
                "priceRange": 200,
                "percentOff": 10,
                "amountOff": 10,
                "limit": 3
           }

  • DELETE Coupons Rule: https://dynamic-cart-api.herokuapp.com/api/coupons/{id}
